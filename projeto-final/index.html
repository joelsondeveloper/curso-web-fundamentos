Programação C++

por Alexander Aronowitz

Aprenda programação com a Linguagem C++.

2ª Edição.

Um guia completo para iniciantes e programadores experientes, cobrindo desde os fundamentos da linguagem até conceitos avançados como ponteiros, orientação a objetos, manipulação de memória e estrutura de dados.

Repleto de exemplos práticos, exercícios desafiadores e dicas valiosas, este livro é ideal para quem quer dominar o C++ e se destacar no mundo da programação de alto desempenho.

Com uma abordagem didática e progressiva, você vai aprender a resolver problemas reais, entender a lógica por trás do código e desenvolver projetos robustos e eficientes.

Padrões JavaScript

por Stoyan Stefanov

Construa aplicações mais robustas usando padrões de projeto e de programação.

O'Reilly · Novatec · Yahoo! Press.

Um guia prático para aprimorar suas habilidades em JavaScript, ensinando como escrever código limpo, modular e de fácil manutenção.

Descubra como aplicar padrões clássicos de design (como Singleton, Factory e Observer) no ecossistema JavaScript moderno, elevando a qualidade dos seus projetos a um novo nível.

Ideal para quem já conhece a linguagem, mas quer escrever código mais profissional, reutilizável e preparado para crescer sem virar um monstro difícil de manter.

Aprenda com exemplos reais e dicas direto de quem já trabalhou no coração do Yahoo!, aplicando técnicas usadas por grandes empresas no mundo todo.
